# GMTK Game Jam 2026

Placeholder repository for the team's game jam project. The project name can be changed once the concept is decided.

## Setup

- Unity `6000.3.20f1` (Unity 6.3 LTS)
- Universal 3D / URP
- WebGL Build Support
- Git LFS

Clone the repository and initialize LFS before opening the project:

```bash
git clone https://github.com/LennonFa/gmtk-game-jam-2026.git
cd gmtk-game-jam-2026
git lfs install
git lfs pull
```

Open the repository folder as a project in Unity Hub. Commit `Assets`, `Packages`, and `ProjectSettings`; generated folders such as `Library`, `Temp`, `Logs`, and `UserSettings` stay local.
